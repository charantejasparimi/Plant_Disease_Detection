# Plant_Disease_Detection
Plant_Disease_Detection using Deep Learning 
We trained the model using Resnet 18 Architecture
Data set is taken from Plant Village Dataset   https://github.com/spMohanty/PlantVillage-Dataset/tree/master/raw/color
Data set contain Total 38 classes of 14 plants
